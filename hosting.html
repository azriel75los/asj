Pengertian Hosting

Hosting adalah layanan yang menyediakan ruang penyimpanan dan akses untuk menyimpan data website di server agar bisa diakses melalui internet.

Fungsi Hosting

  • Menyimpan file website (HTML, CSS, gambar, dll).
  • Menjaga agar website bisa diakses 24 jam.
  • Mendukung email, database, dan layanan lainnya.
